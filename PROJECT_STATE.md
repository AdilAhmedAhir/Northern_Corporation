# Project State

## Status: V1 + V2 + V3 with Premium Hero Experiences

**Last Updated:** 2026-03-10

## Overview

Northern Corporation Ltd. portfolio with A/B/C variation routing — each version features a distinct premium hero experience with scroll-driven animations.

## Variations

### V1: Classic (`/`) — Cinematic Letterbox Reveal
Full-screen factory image seen through cinema bars. On scroll → bars retract, tagline types in character-by-character, Ken Burns slow zoom, image fades as content scrolls up.

### V2: Premium Motion (`/v2`) — Fabric Macro Zoom-Out
Starts at 3x zoom on fabric weave texture. On scroll → zooms out to 1x, text fades in, image pins while content slides over with dark overlay.

### V3: Kinetic Premium (`/v3`) — Radial Mask Reveal
Pitch black screen with pulsing amber dot. On scroll → circular clip-path expands revealing atelier image with mouse parallax. Sticky curtain sections.

### V4: Demo Template (`/v4`) — Legacy Alpha Mask Port
Ported cinematic HTML/CSS/JS template with Alpha Mask statements, horizontal track infrastructure layout, and legacy stats. Cleanly scoped away from global styles.
